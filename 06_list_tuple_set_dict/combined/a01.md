# Aufgabe: Anwendung von Listen

- Erstellen Sie eine Liste mit dem Namen "students", die die Namen von fünf Schülern enthält.
- Erstellen Sie ein leeres Wörterbuch mit dem Namen "grade".
- Geben Sie für jeden Schüler eine Note ein und speichern Sie diese im Wörterbuch "grade", wobei der Name des Schülers als Schlüssel und die eingegebene Note als Wert verwendet wird.
- Berechnen Sie den Durchschnitt der Noten und geben Sie diesen aus.
- Geben Sie die Namen der Schüler aus, die eine Note unter dem Durchschnitt haben.
