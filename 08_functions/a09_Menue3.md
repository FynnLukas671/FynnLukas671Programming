# Aufgabe: Menü 2.0

- Nutzen Sie Ihr Menü aus den vorigen Aufgaben.
- Verändern Sie Ihr Programm nun so, dass die Logik der Menü-Funktionen in eigene Funktionen ausgelagert ist.

```
Zahl ist größer 0!
```
